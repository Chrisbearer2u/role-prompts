1.  Install Python (latest stable version, preferably 3.9+).

            STEP 1- run "python3 -m venv .venv" (creates virtual environment)

            STEP 2- run "source venv/bin/activate" (activates it)

            STEP 3- pip install jupyter numpy pandas

            STEP 4- pip freeze >> requirements.txt

            STEP 5- pip install -r requirements.txt

2.  Verify installation by running:
        
        python --version

    Output:
        
        Python 3.12.3 (main, Jan 22 2026, 20:57:42) [GCC 13.3.0] on linux
        

3.  Write and execute a Hello, World! program.

    Run "python3" on your terminal.

    The terminal will switch to python environment with the default Python prompt of the interactive shell denoted by ">>>"

    Run "print("Hello, World!")"

    Output: "Hello, World!"
